# Termux-Kali NetHunter 安装指南与常见问题

---

## ✨ 有什么用？

Kali NetHunter 是 Kali 官方推出的移动渗透测试平台，支持 200+ 网络安全工具，可在 Android 上运行，适合学习与测试。

## 🚀 如何安装？

### 第一步：安装 Termux

* 官方网站：[https://termux.dev/cn/](https://termux.dev/cn/)（无需魔法）
* 推荐使用：**ZeroTermux**（更兼容更稳定）

### 第二步：下载安装脚本

```bash
# 海外用户（需代理）：
curl -LO Dong-Jing-Yu.github.io/Termux/install-NH

# 中国用户：
curl -LO gitee.com/Dong-Jing-Yu/Dong-Jing-Yu.gitee.io/raw/main/Termux/install-NH
```

* 下载完成后用 `ls` 命令确认文件存在

### 第三步：执行安装

```bash
chmod +x install-NH && ./install-NH
```

* 安装时请选择 **选项 2**（国内镜像更稳定）
* 安装完成后输入 `nh` 启动 Kali NetHunter

## ❓ 常见问题解答（重点强调：**解压问题最多**）

### 1️⃣ `ls` 看不到下载的脚本？

* 说明下载失败或中断
* ✅ 解决：换用国内地址重新下载

### 2️⃣ 下载速度慢？

* 一般是网络问题
* ✅ 解决：换网络或使用 ZeroTermux

### 3️⃣ **最常见！解压过慢或未完成**

* 🚨 90% 安装失败都出在这里！

* 原因：手机性能不足、被后台挂起、网络波动中断下载等

* ❗表现：

  * 解压过程卡住很久
  * `nh` 报错启动失败
  * 命令行提示符不是 Kali 样式

* ✅ 解决方案：

  1. 解压时**保持前台运行**，不要锁屏或切应用
  2. 网络要稳定，避免断网或掉速
  3. 安装失败后请完全删除并重新执行脚本

### 4️⃣ `nh` 命令报错？

* 原因通常是上一步解压失败
* ✅ 解决：重新完整解压一遍，不要跳过

### 5️⃣ Kali 样式没出现？

* 多半是解压没完成或没初始化好
* ✅ 解决：回到解压那一步，耐心再来一次！

### 6️⃣ 工具不完整或缺少组件？

```bash
sudo apt install kali-linux-everything
```

* ⚠️ 警告：请勿随意更改系统文件！

### 7️⃣ 网络扫描类工具无法使用？

* 原因：安卓缺少完整 Linux 权限（如 `/sys`）
* ✅ 解决：建议此类操作用电脑端 Kali 执行

### 8️⃣ 不知道怎么使用 Kali NetHunter？

* ✅ 建议：B 站、网易云课堂、慕课网都有免费教程
