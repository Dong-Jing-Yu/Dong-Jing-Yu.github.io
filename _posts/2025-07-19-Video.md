## Termux

- [x] NapCat
{% include embed/bilibili.html id='BV13jFjz2EN5' %}

  - termux换源\
  `termux-change-repo`

  - 安装proot-distro\
  `pkg install proot-distro`

  - 安装debian\
  `pd i debian`

  - 登录debian\
  `pd login debian`

  - debian:换源\
  `bash <(curl -sSL https://linuxmirrors.cn/main.sh)`

  - 安装必要的包\
  `apt install -y sudo curl libgcrypt20`

  - 下载脚本\
  `curl -O https://raw.githubusercontent.com/NapNeko/NapCat-Installer/refs/heads/main/script/install.sh`

  - 给予权限并运行脚本\
  `chmod -x install.sh && ./install.sh`

  - 创建快捷指令\
  `echo "alias NC='xvfb-run -a /root/Napcat/opt/QQ/qq --no-sandbox'" >> ~/.bashrc`
