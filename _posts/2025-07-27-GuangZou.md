## 🏷️ 基本信息

* **应用名称**：光奏
* **英文名**：GuangZou
* **包名**：`com.djy.guangzou`
* **平台**：Android
* **开发语言**：Kotlin
* **最低 SDK**：API 28（Android 9.0）
* **构建脚本语言**：Kotlin DSL(`build.gradle.kts`)
* **使用Root**：否
* **关于权限授权**：无障碍,存储,悬浮窗
* **琴谱**：琴谱全部为外部 JSON 文件(可以存放在`/storage/emulated/0/光奏/琴谱/`中)
* **琴键**：共 15 个键
* **弹奏方式**：模拟键盘/模拟触控

**琴键布局：**
```
第一排：0  1  2  3  4
第二排：5  6  7  8  9  
第三排：10 11 12 13 14
```

**按键布局：**
```
第一排：Y  U  I  O  P
第二排：H  J  K  L  ;  
第三排：N  M  ,  .  /
```

#### 📄 JSON 琴谱文件格式示例

```json
{
  "title": "小星星",
  "description": "一首简单的入门曲",
  "notes": [
    { "key": [0], "delay": 0 },
    { "key": [0], "delay": 500 },
    { "key": [7], "delay": 500 },
    { "key": [7], "delay": 500 },
    { "key": [9], "delay": 500 },
    { "key": [9], "delay": 500 },
    { "key": [7], "delay": 1000 },
    { "key": [5, 9], "delay": 1000 }
  ]
}
```

#### 字段说明：

| 字段名           | 类型       | 说明                           |
| ------------- | -------- | ---------------------------- |
| `title`       | `string` | 乐曲标题                         |
| `description` | `string` | 简要描述，选填                      |
| `notes`       | `array`  | 音符数组，描述每次点击事件                |
| `key`         | `int[]`  | 琴键编号列表（支持一个或多个同时按下），范围：0\~14 |
| `delay`       | `int`    | 毫秒延迟（与上一个音符之间的时间间隔）          |
