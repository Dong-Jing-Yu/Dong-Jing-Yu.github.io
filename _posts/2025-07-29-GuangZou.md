一个基于 **Kotlin** 开发的 Android 音乐自动弹奏工具，支持无障碍与 ADB 模拟触控，无需 Root 权限

---

## 📱 应用信息

| 属性           | 值                                         |
|----------------|--------------------------------------------|
| **应用名称**   | 光奏                                        |
| **英文名**     | GuangZou                                   |
| **包名**       | `com.djy.guangzou`                         |
| **平台**       | Android                                    |
| **开发语言**   | Kotlin                                     |
| **最低 SDK**   | API 28（Android 9.0）                      |
| **构建工具**   | Kotlin DSL (`build.gradle.kts`)            |
| **权限**   | 无障碍,存储,悬浮窗                              |
| **ADB工具**    | Shizuku                                    |
| **Root**  | ❌                                             |

---

## 🔐 权限说明

- **无障碍服务**：用于模拟触控弹奏
- **存储访问**：读取/写入琴谱文件
- **悬浮窗**：用于浮动控制面板

---

## 🎼 琴谱存储

- 路径：`/storage/emulated/0/光奏/琴谱/`
- 支持格式：`.json`(推荐)
- 适配**未加密**的**sky studio**琴谱文件(`.txt`)

```json
{
  "name": "标题",
  "author": "作者",
  "bpm": 320,
  "notes": [
    { "time": 0, "key": [1,2,3] },
    { "time": 1000, "key": [9] }
  ]
}
```

**key对应的琴键：**
```
第一排：0  1  2  3  4
第二排：5  6  7  8  9
第三排：10 11 12 13 14
```


---

## 🎹 弹奏机制

- **核心方式**：模拟触控
- **实现手段**：
  - 无障碍 API
  - `adb shell input swipe X Y X Y 0`
- **坐标策略**：
  - 优先使用预设分辨率布局
  - 若无匹配布局 → 选取相近的布局 → 自行设置琴键坐标(只需要四个角的坐标即可)
